# 🌐 Portfólio Pessoal - Landing Page HTML & CSS

Projeto de desenvolvimento de uma **landing page de portfólio pessoal**, com foco em apresentação profissional, design moderno e navegação intuitiva.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

---

## 🎯 Objetivo

Praticar a criação de páginas completas para apresentação pessoal, incluindo:

* Estruturação de layout de site
* Criação de navbar (menu de navegação)
* Organização de conteúdo principal
* Aplicação de estilos modernos

---

## ⚙️ Funcionalidades

* ✔️ Menu de navegação com links
* ✔️ Seção de apresentação pessoal
* ✔️ Botões de ação (Login / Registrar / Contrate-me)
* ✔️ Layout centralizado e organizado
* ✔️ Background com imagem

---

## ✨ Diferenciais técnicos

* Estrutura de **landing page profissional**
* Uso de **Flexbox** para alinhamento
* Design moderno com destaque em cores
* Botões com efeito visual interativo
* Código limpo e de fácil manutenção
